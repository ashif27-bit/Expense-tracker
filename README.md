# 💰 Expense Tracker

A simple and responsive Expense Tracker built using **HTML**, **CSS**, and **JavaScript**.

## 📌 Features

- **🌓 Automatic & Manual Dark Mode**: Fluid theme transitions that respect user preference and save your choice instantly using local storage.
- **📊 Real-time Summaries**: Dynamically calculates your current balance, total income, and total expense values in real-time.
- **🔍 Smart Search & Filter System**: 
  - Search transactions instantly by description or category.
  - Filter transactions by type (*All*, *Income*, *Expense*).
  - Sort entries seamlessly by *Newest*, *Oldest*, *Highest*, or *Lowest* amounts.
- **📅 Periodical Reporting**: Automated breakdowns of your financial habits, giving you clear insights into your **Monthly** and **Yearly** net margins.
- **💾 Local Storage Persistence**: Your data is saved locally on your browser. Refreshing or closing the tab won't lose your ledger history.
- **📱 Fully Responsive**: Optimized for every screen size—from extra-large desktop monitors down to mobile screens.

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
