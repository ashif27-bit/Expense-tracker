# 💰 Expense Tracker

A simple and responsive Expense Tracker built using **HTML**, **CSS**, and **JavaScript**.

## 📌 Features

- Add Income
- Add Expense
- View Current Balance
- Transaction History
- Delete Transactions
- Data saved using Local Storage
- Mobile Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
